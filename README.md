# Instalador Automático de LibreNMS para Rocky Linux

Script de instalación automatizada para desplegar LibreNMS en Rocky Linux 8.10. 

Este script aprovisiona todos los componentes necesarios, incluyendo MariaDB, Nginx, PHP-FPM, SNMP y dependencias del sistema, aplicando configuraciones específicas para la distribución con el fin de garantizar un entorno consistente y reproducible.

Ideal para hombrelabs y entornos educativos.

No usar en entornos de producción.
